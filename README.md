## Requirements

Zsh 5.x

## Install

```zsh
git clone git://github.com/andsens/homeshick.git ~/.homesick/repos/homeshick
~/.homesick/repos/homeshick/bin/homeshick clone herrbischoff/dotfiles
source ~/.zshrc
```
